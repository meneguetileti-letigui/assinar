☁ Configurar Supabase
Passo 1 de 4 — Abrir o Supabase
O Supabase é um banco de dados gratuito na nuvem. Você já tem conta. Vamos apenas pegar as credenciais do seu projeto.
1
Abra supabase.com e faça login na sua conta
2
Clique no seu projeto existente (ou crie um novo com New Project)
3
No menu lateral esquerdo clique em Project Settings (ícone de engrenagem ⚙)
4
Clique em API no submenu
Passo 2 de 4 — Copiar as credenciais
Na página Project Settings → API você vai ver dois valores. Copie e cole abaixo:
📌 Em Project URL — copie a URL que começa com https://
📌 Em Project API keys — copie o valor de anon public (a chave longa que começa com eyJ)
Passo 3 de 4 — Criar as tabelas
No Supabase, clique em SQL Editor no menu lateral → clique em New query → cole o código abaixo → clique em Run.
create table if not exists asc_dados ( id text primary key, atletas text, contratos text, next_aid int default 1, next_cid int default 1, atualizado_em timestamptz default now() ); create table if not exists asc_assinaturas ( token text primary key, contrato_id text, status text default 'empresa_assinou', nome_atleta text, email_atleta text, tipo text, prazo int, data_contrato text, assinado_em timestamptz, criado_em timestamptz default now() ); alter table asc_dados enable row level security; alter table asc_assinaturas enable row level security; create policy "acesso_total_dados" on asc_dados for all using (true) with check (true); create policy "acesso_total_assin" on asc_assinaturas for all using (true) with check (true);
Passo 4 de 4 — Testar e salvar
Clique em Testar conexão. Se aparecer ✓, clique em Salvar e conectar.
